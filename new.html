Preview:

Program is created for Civil Engineers to analyze roadway. Main goal is to process data from laser scanner TiM 551 - SICK. All information that we need to get are compute in SOPAS engeenring tool and save in csv or excel file. File generete from SOPAS contain state device during measure and data of measure. Data from device looks :

Angle1 Angle2 Angle[..]
measure1 distance1_1 distance1_2 distance1_[..]
measure2 distance2_1 distance2_2 distance2_[..]

Program features:

  • reading data
  • cleaning and preparation data
  • data processing
  • data aggregation and operations performed on groups
  • graphs and data visualization
  • saving calculated data
